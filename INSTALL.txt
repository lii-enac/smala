We provide a Makefile that should work on MacOS (10.12+),  Linux platforms, windows 10+

---- LINUX -----

Linux/Ubuntu:
  git clone git@github.com:lii-enac/smala.git
  cd smala
  make install-pkgdeps
  make -j4

---- MACOS -----

MacOS
  git clone git@github.com:lii-enac/smala.git
  cd smala
  make install-pkgdeps
  make -j

---- WINDOWS -----

Win
  launch MINGW64 command line then
  git clone git@github.com:lii-enac/smala.git
  cd smala
  make install-pkgdeps
  make -j4
